# Телепромптер

Синхронный телепромптер для 3 устройств. Один контроллер, два дисплея.

## Деплой на Railway

1. Создай репозиторий на GitHub, загрузи все файлы
2. Зайди на [railway.app](https://railway.app) → New Project → Deploy from GitHub repo
3. Выбери репозиторий → Railway автоматически всё задеплоит
4. В настройках проекта: Settings → Networking → Generate Domain
5. Открывай ссылку на всех трёх iPhone в Safari

## Использование

- **Контроллер** — вставь текст, нажми «Отправить», управляй кнопками
- **Дисплей** — просто открой ссылку и выбери «Дисплей»
- ⏮ / ⏭ — перемотка на экран назад/вперёд
- Слайдер скорости — от 10 до 300 px/сек
- Слайдер размера — от 24 до 120px
- Полоска позиции — тап в любое место для быстрого перехода

## Локальный запуск (опционально)

```bash
npm install
npm start
```
